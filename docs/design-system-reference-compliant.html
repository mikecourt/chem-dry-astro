Typography

📝 Typography Notes:

We use Poppins for headlines (modern, friendly) and Inter for body text (highly readable). Fallback to Arial/Futura for Chem-Dry brand compliance where needed.

Body Text Color: Use #444444 for optimal readability. The Chem-Dry Warm Grays (#A29588, #8A7967) are too light for body text and should only be used for backgrounds or subtle accents.

Heading 1 - Poppins Bold 48px

Used for: Hero headlines, main page titles

Heading 2 - Poppins Bold 36px

Used for: Major section headings

Heading 3 - Poppins Semibold 30px

Used for: Subsection headings, card titles

Heading 4 - Poppins Semibold 24px

Used for: Component titles, feature headlines

Heading 5 - Poppins Semibold 20px

Used for: Small headers, list titles

Body text - Inter Regular 16px. This is the standard body text used throughout the site. It's highly readable and optimized for screen viewing. The line height is set to 1.6 for comfortable reading.

Large body text - Inter Medium 18px. Used for emphasis or lead paragraphs that need to stand out.

Small text - Inter Regular 14px. Used for disclaimers, captions, and secondary information.

Buttons

Button Usage Guidelines

  • Primary (Green): Main actions like "Schedule Now", "Get Quote", "Book Service"
  • Secondary (Navy): Secondary actions like "Learn More", "View Services", "See Details"
  • Outline: Less emphasis actions, alternative options, tertiary CTAs
  • CTA (Gradient): Hero buttons, phone numbers, highest priority conversion actions

Accessibility Standards

  • Minimum target size: 44x44px for mobile
  • WCAG AA contrast ratio compliance (4.5:1 minimum)
  • Focus states visible and clear
  • Keyboard navigable

Chem-Dry Brand Guidelines

⚠️ CRITICAL: Never Break These Rules
  • Never use "Chem-Dry's" (possessive)
    ❌ "Chem-Dry's process" → ✅ "The Chem-Dry process" or "The process used by Chem-Dry"
  • Always include modifiers with water usage claims
    ❌ "uses 80% less water" → ✅ "uses about 80% less water"
  • Never say "no residue"
    ❌ "no residue" → ✅ "no dirt-attracting residue"
  • The Natural® - proper capitalization and ®
    ❌ "the natural" → ✅ "The Natural®"
  • Never say "all natural" or "green certified"
    These phrases are prohibited by brand guidelines

✅ Approved Phrases - Use These:

  • "The process used by Chem-Dry..."
  • "The method used by Chem-Dry..."
  • "Hot Carbonating Extraction"
  • "The Natural® uses carbonation for a better clean"
  • "uses about 80% less water" (always include modifier)
  • "no dirt-attracting residue" (never just "no residue")
  • "The Natural® doesn't rely on soapy detergents or harsh chemicals"

Voice & Tone Guidelines

  • Professional but human, friendly, and familiar
  • Write like a real person, not a corporation
  • Use contractions (we're, you'll, it's)
  • Clear and concise - no fluff
  • Consumer-facing language, not technical jargon
  • Active voice preferred

SEO & Technical Standards

Performance Targets

  • PageSpeed Score: 90+ on mobile and desktop
  • Largest Contentful Paint (LCP): < 2.5s
  • First Input Delay (FID): < 100ms
  • Cumulative Layout Shift (CLS): < 0.1
  • Image Format: WebP with fallbacks
  • Image Compression: Optimized, lazy-loaded

Schema Markup Requirements

  • LocalBusiness schema on all city pages
  • Service schema for each service offering
  • Review/AggregateRating schema for testimonials
  • BreadcrumbList for navigation
  • FAQPage schema for FAQ sections

Meta Tags Best Practices

  • Title: 50-60 characters, include city + service
  • Description: 150-160 characters, include USP and CTA
  • H1: One per page, include primary keyword
  • Alt Text: Descriptive, keyword-relevant, not stuffed
  • Canonical Tags: Prevent duplicate content issues

Mobile Responsiveness

Breakpoints

Mobile: 320px - 767px
Tablet: 768px - 1023px
Desktop: 1024px - 1279px
Large Desktop: 1280px+

Mobile-First Principles

  • Design for mobile first, enhance for larger screens
  • Touch targets minimum 44x44px
  • Readable text without zooming (16px minimum)
  • Single-column layouts on mobile
  • Sticky header with click-to-call on mobile
  • Forms optimized for mobile input
